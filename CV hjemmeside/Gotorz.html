3. semesters projekt i Web-Development – Gotorz

Gotorz var vores 3. semester projekt på datamatikeruddannelsen, med fokus på Web-Development. Projektet fungerede både som en læringsplatform til at afprøve ny viden fra undervisningen og som det eksamensprojekt, vi skulle vurderes på.

Projektet er en Full-Stack webapplikation udviklet til en kunde i Dubai, som er en rejseudbyder, der henvender sig til Scandinaviske og Europæiske kunder.

Jeg har primært arbejdet med Back-end og DevOps-delen af løsningen.

Projektet består af tre separate komponenter:

  • Gotorz: En Blazor Web App, som udgør selve frontend-applikationen.
  • UserAndAuthAPI: En ASP.NET Core Web API til brugerautentifikation.
  • TravelBridgeAPI: En ASP.NET Core Web API som fungerer som et cache-lag mellem Gotorz og Booking.com API. Den vælger cyklisk mellem 2–4 API-nøgler for at balancere forespørgsler.

Hele løsningen er containeriseret med Docker og deployet på en privat Ubuntu-server ved hjælp af Docker Compose og NGINX som reverse proxy.
